# Demo 流程

## 展示目標

讓老師看到這是一個完整的 Python 資訊系統：有使用者、資料查詢、收藏、筆記、資料庫保存。

## Demo 步驟

1. 啟動系統

```bash
cd /Users/weishan33/Desktop/Coffee_Run!!/coffee_run_streamlit
python3 -m streamlit run app.py
```

2. 輸入暱稱登入

範例：`Ally`

3. 探索咖啡店

- 搜尋「安靜」
- 選擇地區「公館」
- 調整最低評分

4. 收藏咖啡店

- 收藏一間店
- 切到「我的收藏」
- 確認收藏出現

5. 新增咖啡筆記

- 切到「咖啡筆記」
- 選一間店
- 輸入心得
- 確認筆記出現在下方

6. 展示資料永久保存

- 重新整理網頁
- 用同一個暱稱登入
- 收藏與筆記仍然存在

7. 推薦排行

- 切到「推薦排行」
- 說明系統依照評分排序

## 講解重點

- 全部功能由 Python 程式完成
- 使用 Streamlit 做介面
- 使用 SQLite 保存資料
- 專案有清楚分工與文件
