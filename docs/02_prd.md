# Coffee Run 產品需求文件

## 1. 專案背景

學生常常需要找適合讀書、討論或短暫休息的咖啡店，但一般地圖工具資訊分散，不容易快速比較咖啡店是否適合學生日常使用。Coffee Run 希望用簡單的資訊系統幫使用者搜尋咖啡店、收藏店家、記錄心得，並查看推薦排行。

## 2. 目標使用者

- 想找咖啡店讀書的大學生
- 想和朋友找聚會地點的學生
- 想記錄自己去過咖啡店的人

## 3. 核心價值

- 快速找到符合需求的咖啡店
- 收藏想去或喜歡的店
- 用筆記記錄每次咖啡體驗
- 透過排行快速看到推薦店家

## 4. MVP 功能

### 4.1 簡易登入

使用者輸入暱稱即可進入系統。這是課堂專案用的假登入，不處理密碼與驗證。

### 4.2 探索咖啡店

使用者可以看到咖啡店清單，並依照關鍵字、地區、最低評分篩選。

### 4.3 收藏咖啡店

使用者可以收藏或取消收藏咖啡店。收藏資料會永久存進 SQLite。

### 4.4 咖啡筆記

使用者可以針對咖啡店新增評分與心得。筆記會永久存進 SQLite。

### 4.5 推薦排行

系統依照店家評分排序，顯示推薦咖啡店。

## 5. 非功能需求

- 使用 Python 開發
- 使用 Streamlit 建立介面
- 使用 SQLite 保存資料
- 不自行撰寫 HTML、CSS、JavaScript
- 可以在本機啟動與展示

## 6. 暫不做的功能

- 真實 Google Maps API
- 真實 GPS 定位
- Firebase 登入
- 線上付款
- 密碼登入
- 多人即時聊天

這些功能不是 MVP 必需項目，會增加技術風險，也可能違反只用 Python 的作業限制。
